GAWK

awk的GNU版本，提供了一种编程语言。

  1. 定义变量保存数据；
  2. 使用算数和字符串操作处理；
  3. 结构化编程
  4. 格式化。

格式及基础

选项 描述
-F fs 指定行划分数据字段的分隔符
-f file 指定文件读取
-v var=value 定义变量及默认值
-mf N 指定处理数据文件最大字段数
-mr N 指定处理数据文件最大行数
-W keyword 指定gawk的兼容模式和警 告等级

如，重复输出”Hello, world!”：

gawk '{print "Hello, world!"}'

control + D返回

另外：$0、$1、$2、$3…分别表示整个文本行和文本行的第1、2、3个字段；如：

echo "My name is Amy~" | gawk '{$4="Tony"; print $0}'

GAWK进阶

变量

数据字段变量：$0,$1,$2,$3,$4…

变量 描述
FS 输入字段分隔符
RS 输入记录分隔符（默认换行符）
OFS 输出字段分隔符
ORS 输出记录分隔符（默认换行符）
FIELDWIDTHS 有空格分隔一段数字，定义了每个数据字段的确切宽度

其他数据变量（使用不需要加$符号）：

数据变量 描述
ARGC 当前命令行参数
ARGIND 文件在ARGV中的位置
ARGV 命令行参数的数组
CONVFMT 数字的转换格式，默认为%.6 g
ENVIRON 当前SHELL环境变量及值组成的数组
ERRNO 读取或关闭输入文件发生错误的系统错误号
FILENAME 输入数据的文件名
FNR 数据文件行数（多个文件会重置）
IGNORECASE 非零值，将忽略大小写
NF 数据文件字段总数
NR 已处理输入记录数
OFMT 数字输出格式，默认为%.6 g
RLENGTH 由match函数所匹配的子字符串长度
RSTART 由match函数所匹配的子字符串的起始位置

数组与匹配

gawk采用关联数组来定义，即：散列表&字典。

匹配操作符，匹配限定记录字符串：$1 ~ /^data/

命令

和c语言类似。

if语句

格式： if (condition) statement

while语句

格式 while (condition) {statement}

注意，为了区分语句块，请多用{}符号

do-while语句
for语句

格式：for () {}

打印

printf和c一样。

控制字母 描述
c 一个数作为ascii字符
d 显示一个整数
i 显示一个整数（与d一样）
e 科学计数法显示
f 显示浮点数
g 用科学计数法或者浮点数显示（较短的格式）
o 八进制
s 文本字符串
x 16进制
X 16进制，但是用字母A-F

函数

数学函数 描述
atan2(x,y) 反、正切
int(x) 取整数靠近0一部分
cos(x) 余弦
exp(x) 指数函数
rand( ) 0-1随机数
sin(x) 正弦
sqrt(x) 平方根
log(x) 自然对数
srand(x) 随机种子值
字符串函数 描述
asort(s [, d]) 数组s按照数据数值排序，若指定d，则存储在d中
asorti(s [,d]) s按照索引值排序，若指定d，则存储在d中
gensub(r,s,h[,t])
gsub(r,s[,t])
index(s,t) 返回t在s中的索引值，没有则返回0
length([s]) 返回字符串长度
match(s,r[,a]) s中正则表达式出现未知的索引，如果指定a，a存储匹配正则表达式部分
split(s,a[,r]) 将s用FS字符或者正则表达式r分开存储在a中
sprintf(format,variables) 用format和variables返回类似于printf的字符串
sub(r,s[,t]) 在变量$0或目标字符串t中查找正则r的匹配，找到了用s替换
substr(s,i,[,n]) 返回s中从索引i开始n个字符组成的字符串。未提供n则返回剩下部分
tolower(s) 所有字母转化为小写
toupper(s) 所有字幕转化为大写
时间函数 描述
mktime(datespec)
strftime(format[,timestamp]) 转化为日期格式
systime() 返回当前时间戳值

自定义函数（必须出现在所有代码块之前）

function name ([variables]){statements}

使用库函数，直接加上参数-f即可。
