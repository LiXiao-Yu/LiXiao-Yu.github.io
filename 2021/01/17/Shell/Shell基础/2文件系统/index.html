文件权限

linux系统安全性

用户密码在/etc/passwd下，UID在500以下均为系统预留账户。

从500开始，系统会将第一个可用UID给用户。

/etc/passwd格式如下：

登录用户名 密码 用户账户的UID 组UID 用户文本描述 HOME 默认shell
bin x 1 1 bin /bin /sbin/nologin
daemon x 2 2 daemon /sbin /sbin/nologin
adm x 3 4 adm /var/adm /sbin/nologin

密码均为x，密码文件在/etc/shadow下，共9项。

分别代表：

用户名、加密密码、上次修改密码后天数（1970/01/01算起）

多少天后才能修改密码、多少天后必须修改密码、过期前多少天提醒用户更改密码

密码过期多少天禁用账户、用户账户被禁用日期（1970/01/01算起）、预留字段

用户

添加用户：

useradd -D 查看新增用户默认值

useradd -b -e -f -g -s 修改默认值

useradd … 创建新用户，修改值

useradd -m 创建新用户，并把/etc/skel目录中，bash登录文件复制到新用户HOME目录下

删除用户：

userdel -r：默认删去/etc/passwd下文件信息，不会删除系统所属文件；-r 会删除HOME及邮件目录

修改用户：

usermod ：可修改/etc/passwd全部字段；-L将用户锁定，无法登陆；-U恢复正常

passwd：修改密码，-e 强制下次登录更改密码

chpasswd：修改密码，可重定向

chage、chfn、chsh：修改特定账户信息

组

组文件保存在/etc/group文件中

组名 组密码 GID 用户列表
root x 0 默认组不会列出用户

不允许直接修改/etc/group修改用户组名，推荐使用usermod命令

usermod -G shared daemon：为shared加入daemon用户

创建新组：groupadd

修改组：groupmod

文件权限

文件所属主权限 文件属组权限 其他用户权限
rwx r-x r-x

创建文件默认权限：umask

一般是输出0022（掩码），代表了644权限；

不看第一位，文件的全读写权限为666，减去022，即666-022=644

对于目录来说，目录全读写执权限为777，减去022，为755。

改变权限：

chmod 755；

chmod o+r\u-x；第一个字符u(用户)g(组)o(其他)a(所有)；+/-增加减少权限；r/w/x。

改变所属关系：chown ：chown user.group XXX

chgrp更改文件默认属组， change group1 file。

PS：linux基本权限位为9位，但是还有额外的三个权限为，分别为：SUID、SGID、黏着位。

涉及命令

  • useradd -m
  • userdel -r
  • usermod -L/-U
  • groupadd
  • groupmod
  • umask
  • chmod a-w ：所有用户减去写权限
  • chown
  • chgrp

文件系统（File System）

ext文件系统

ext文件系统：extended filesystem，虚拟目录操作硬件；定长块存储数据。

ext1包含文件名、文件大小、文件属主、文件属组、文件权限、文件指针，linux通过文件索引访问。

ext2新增时间值，按组分配磁盘，减少磁盘碎片化管理。

ext1和ext2直接写入磁盘。

日志文件系统（类似于用日志做了一个缓存）

现将文件更改写入临时文件（日志）中，在数据成功写入存储以及索引节点表后，删除日志目录。

操作文件系统

创建分区： fdisk /dev/vda；

进入fdisk交互式cli中，p列出当前分区；n创建新的分区……

格式化文件系统

mkfs.xfs：创建一个xfs文件系统

mkfs.ext4：创建一个ext4文件系统

将分区挂载

mount （-t ext4） /dev/vda /mnt/my_partition

错误检查： fsck

涉及命令

  • lsblk
  • fdisk
  • mkfs.ext4
  • mount

安装程序

在此只介绍yum，不介绍apt。

查看可安装软件名，此处可以进行模式匹配：yum list ***

查看某个文件属于哪个yum包：yum provides ***

安装： yum install ***

本地安装：yum localinstall *** （rpm -ivh 区别？）

列出可用更新：yum list updates

更新某个软件：yum update （***）

删除软件，保留配置文件和数据文件：yum remove ***

删软件所有信息：yum erase ***

确认安装软件包的依赖关系：yum deplist ***
