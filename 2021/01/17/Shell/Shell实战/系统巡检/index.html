系统巡检

任务要求

需要对 Linux 服务器，巡检系统 / 服务 / 端口各指标及参数，编写系统快速检查脚本，输出系统信息到脚本运行的 logs 目录下。

脚本

涉及命令

命令 描述
id -u 当前用户的UID
cut -d - f3 以-作为分隔符号，然后获取第3块
date date +”%Y-%m-%d”<br /
%Y：年全称，如：2020
%y：年缩称，如：20
%m：月
%d：天
%H：小时
%M：分钟
%S：秒
%h：月简称
%a：星期简称
grep -e：多选
-v：反选
-w：只选择匹配行
-f：从文件中选择
-i：忽略大小写
uname -m：获取架构信息
-o：获取系统信息
-r：内核信息
